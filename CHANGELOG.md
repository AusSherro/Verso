# Changelog

All notable changes to Verso will be documented in this file.

## [1.0.0] — 2026-03-16

### Added
- Initial release
- Full reading stats dashboard: library overview, reading time, currently reading, monthly completions, top books by time, top authors, recently finished, annotations, library composition, day-of-week patterns, yearly summary
- Bundled statically-compiled ARM `sqlite3` binary (musl libc) — no system dependencies required
- NickelMenu integration with one-tap launch
- E-ink-optimized HTML/CSS styling
- Error pages shown in browser instead of cryptic NickelMenu errors
- POSIX shell script — runs on any Kobo with firmware 4.x
